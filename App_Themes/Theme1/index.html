June 01, 2010

Welcome to "falling leaves"...

...also known as "andreas06", another free XHTML/CSS template by Andreas. This one has a lot of useful features, such as the multi-level menu and a useful set of formatting and layout styles. The simple code structure makes it easy to customize the look and feel of the design, and several layout variations are included. Click the menu tabs to see examples of them.

Like all my free website templates, andreas06 is completely free to download, free to modify in any way you may want to and free to use without any obligations or limitations. I kindly ask you to leave the design credit link in the footer since it is a great way of supporting my template design work, but it is no requirement.

An award-winning design

The template was originally created as an entry in an international CSS template design contest on OpenWebDesign in October 2005. The contest theme was "fall/autumn", and I built the template design around quality code and colors that I extracted from a photo taken on a beautiful September in a park close to my home. The andreas06 template was awarded 1st prize and the title "OWD Fall/Autumn Contest Winner"!
